# Labalaba — User Documentation

**A modern, cross-platform process manager with a glassmorphism desktop UI.**

Spawn, monitor, and control any program — binaries, scripts, or Python apps — from one beautiful desktop window. Labalaba keeps your tasks running in the background while you stay in control.
