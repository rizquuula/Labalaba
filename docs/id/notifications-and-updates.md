# Notifications & Updates

Tetap terinformasi saat task memerlukan perhatian dan pastikan Labalaba selalu diperbarui.

---

## Notifikasi desktop

Labalaba dapat mengirim notifikasi desktop native setiap kali sebuah task crash atau berhenti secara tidak terduga, sehingga Anda tidak perlu terus membuka jendela aplikasi untuk mengetahui bila ada yang tidak beres.

**Untuk mengaktifkan atau menonaktifkan notifikasi:**

1. Buka Settings (gear icon di bilah atas).
2. Masuk ke bagian **Notifications**.
3. Aktifkan atau nonaktifkan **Desktop Notifications**.
4. Klik **Save Settings**.

Saat notifikasi aktif, Anda akan menerima notifikasi setiap kali sebuah task:

- Keluar dengan kode non-zero (error) — task ditampilkan sebagai **crashed**.
- Berhenti dengan cara tidak terduga lainnya.

> **Catatan:** Anda tidak akan mendapat notifikasi saat menghentikan task secara manual — notifikasi hanya untuk penghentian yang tidak terduga.

## Tetap diperbarui

### Pemeriksaan pembaruan otomatis

Saat **Auto-check for Updates** diaktifkan (default), Labalaba secara diam-diam memeriksa versi baru sekitar sekali setiap 24 jam. Anda tidak perlu melakukan apa pun.

Saat versi yang lebih baru ditemukan, dialog **New Version Available** (versi baru tersedia) akan muncul dan menampilkan:

- Versi Anda saat ini → versi yang tersedia.
- Catatan rilis untuk versi baru (jika tersedia).
- Dua tombol:
  - **Remind Me Later** — menutup dialog; Labalaba akan menampilkannya kembali pada pemeriksaan berikutnya.
  - **Download Update** — membuka halaman rilis GitHub di browser Anda di `https://github.com/rizquuula/labalaba/releases` sehingga Anda dapat mengunduh installer untuk platform Anda.

### Memeriksa pembaruan secara manual

Anda dapat memeriksa kapan saja tanpa menunggu interval otomatis:

1. Buka Settings (gear icon di bilah atas).
2. Masuk ke bagian **Updates**.
3. Klik **Check for Updates Now**.

Hasilnya ditampilkan secara langsung:

- Jika pembaruan tersedia: "Update available: X.Y.Z" beserta tautan **View Release**.
- Jika Anda sudah menggunakan rilis terbaru: "You're on the latest version (X.Y.Z)".

### Menonaktifkan pemeriksaan otomatis

1. Buka Settings.
2. Di bagian **Updates**, nonaktifkan **Auto-check for Updates**.
3. Klik **Save Settings**.

Anda tetap dapat memeriksa secara manual kapan saja menggunakan **Check for Updates Now**.

> **Tip:** Jika pemeriksaan pembaruan gagal, pastikan Anda memiliki koneksi internet yang aktif dan firewall Anda mengizinkan Labalaba untuk mengakses `github.com`. Lihat [Pemecahan Masalah](./troubleshooting.md) untuk informasi lebih lanjut.
